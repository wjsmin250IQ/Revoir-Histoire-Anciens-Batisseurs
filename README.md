# Projet : Réexamen historique des anciens bâtisseurs

## Objectif
Collection organisée de ressources sur les civilisations disparues et l'histoire alternative, notamment la Tartarie, pour :
- Servir de base à des recherches historiques alternatives
- Documenter les théories marginales sur les anciennes civilisations
- Offrir un exemple d'organisation de ressources multimédias

## Structure
- `Donnees_Documents/` : Cartes et images historiques
- `Donnees_Lien_URL/` : Liens organisés par plateforme
- `Archives/` : Documents historiques numérisés

## Méthodologie
Croisement de sources :
- Cartes anciennes
- Archives journalistiques
- Recherches académiques
- Théories alternatives

## Comment contribuer
1. Vérifier les liens brisés
2. Ajouter des métadonnées aux ressources
3. Croiser avec des sources académiques
